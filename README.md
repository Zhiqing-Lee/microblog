#简易微博系统

一个用nodejs写的简单的微博客系统

![microblog](http://i2.buimg.com/3d09f6a3880f15e2.png)

示例： [Microblog](http://mb.zhiqing.info)

用法：

```shell
git clone https://github.com/Zhiqing-Lee/microblog.git
cd microblog
npm install
npm start
```

浏览器访问: http://localhost:3000

setting.js:

```javascript
db: 'db.sqlite3' //数据库文件
```
